<h1>My chatbot template</h1>

-----------------------------------------------

<p><strong> To run: </strong></p>

* Clone this project:

  ```
  git clone git@github.com:alves-Moises/chatbot_template.git
  ```

* Install dependencies:

  ```
  npm install
  ```

* Run:

  ```
  node index.js
  ```

Note: You need the NodeJS to run this project.
